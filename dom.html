DOM (Document Object Model)

Representa el documento HTML en forma de una estructura de arbol.
El documento se puede manipular facilmente a travez del arbol, es decir, agregar, eliminar, modificar elementos etc.

  1. Acceder elementos del documento
    Metodos del documento:
    • document.getElementsByTagName('etiqueta')
    • document.getElementsBy('clase')
    • document.getElementById('id')
  2. Modificar el contenido de un elemento
    Propiedades del elemento:
    • elemento.textContent = 'texto'
    • elemento.innerHTML = 'codigo HTML'
  3. Agregar elementos (nodos) nuevos:
    1. Crear el elemento (createElement())
    2. Crear el contenido (createTextNode())

    4. Agregar atributos (setAtribute())
    5. agregar el nuevo elemento al documento (appendChild())
