Hipertext Markup Language

  • Las páginas HTML son documentos de textos.
  • HTML usa elementos para definir la estructura de la página.
  • Los elementos se definen por etiquetas encerradas en paréntesis angulares.
  • El contenido de un elemento se encierra entre etiquetas.
  • Existen etiquetas de apertura y de cierre.

Las etiquetas pueden tener atributoslos cuales se especifícan en la etiqueta de inicio. Los atributos contienen información adiciomnal acerca del contenido
