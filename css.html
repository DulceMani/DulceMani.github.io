CSS (Cascading Style Sheet)

Versiones de CSS:

  • CSS1 -> 1996(borrador) -> 1999 (liberado) -> 2001 (Soportado por todos los navegadores)
  • CSS2 -> 1998 (borrador) -> 2007
  • CSS3 -> 2001 -> 2013

Reglas CSS

Las reglas permiten cambiar la apariencia (presentación) de los elementos HTML

  • Sintaxis general de CSS
    • selector{declaraciones}
  • declaraciones CSS
    • propiedad:valor;

Tipos de estilo

  1. Por default (navegador).
  2. Del usuario (archivo local)
  3. Hoja de estilo externa
    • Usa la etiqueta <link> para hacer referencia al archivo
  4. Estilo interno <style>
  5. Estilo en linea
    • Se asigna la propiedad y valor.

Ejemplos

Selectores CSS

Selector de tipo
Selecciona un tipo de elemento HTML y aplica el estilo a todos los eleentos del mismo tipo
Ej.
p{text-align:justify;}
Selector descendente
Aplica el estilo a elementos que están contenidos dentro de otros elementos.
Ej.
pb{color: red;}
ul{i b {color: green;}}
Selector hijo
Similar al desendiente pero mas restrictivo aplica el estilo solamente a los elementos que sean hijos directos de otro elemento.
Ej.
p < b {color: blue;}
Ej. HTML
< p << i <<b < Hola < / b< < / i< < / p<
Selector por ID
Un ID es un atributo de un elemento en HTML y se utiliza para aplicar un estilo de manera particular.
Ej.
< p id= "introduccion" < Este es el parrafo de la introduccion </ p<
# instruccion {color: blue;}
Selector por clase
Aplicar estilo a un conjunto de elementos
Ej.
<p class = "especial" > Este es un parrafo especial < / p<
. especial {color: red;}
<p class = "especial"< este es otro parrafo < /p<
Selector universal *
Aplica el mismo estilo a todos lo elementos de la página.
Los estilos se eredan, si aplicas un estilo a un elemento todos los elementos dentro de él tendrán el mismo estilo.
