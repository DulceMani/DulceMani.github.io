Arreglos en JavaScript

Creación

var arreglo = Array()
var arreglo = []

var arreglo = Array('hola',123,true)
var arreglo = ['hola',123,true]

var m = Array(Array(1,2,3),Array(4,5,6))
var m = [[1,2,3],[4,5,6]]

Acceso

var x = arreglo[0]
por lo tanto:
x = 'hola'

arreglo[0] = 'mundo'

Métodos

concat()
a.concat(b) concatena dos arreglos
join()
Unir todos los elementos de un arreglo en cadena a.join() a.join(',')
push()
Agrega un elemento al final del arreglo
pop()
Eliminar el ultimo elemento al final del arreglo
reverse()
Regresa un arreglo con los elementos en orden invertido
sort()
Ordena los elementos de manera ascendente tratandolos como cadena
shift()
elimina del inicio
unshift()
inserta al inicio
splice()
Agregar o eliminar elementos
slice()
extrae en un sub arreglo
indexOf()
posición del elemento del arreglo
forEach()
necesita 3 paramentros, elemento, indice, arreglo dentro de una funcion
