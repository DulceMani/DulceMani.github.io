JQuery

Simplifica:

  • Interaccion con el DOM con HTML
  • La manipulacion DOM
  • La manipulacion CSS
  • Manejo de eventos
  • Animaciones
  • Interacciones AJAX

Garantiza la compatibilidad con los navegadores.
Todo se realiza atravez de una funcion llamadi jQuery().

Recibe como parametro una expresion CSS y regresa un objeto jQuery.

var objeto = jQuery(selector)


Tipos de selectores

  • *
  • elemento
  • #id
  • .clase
  • sel1, sel2

Filtros basicos

  • :first
  • :last
  • :even
  • :odd
  • :eq (index)
